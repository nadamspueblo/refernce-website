Random Numbers

Notes

Intro

Random numbers are useful in software development for many reasons - games mechanic, generating usernames/passwords, any time you need something seemingly unpredictable
Pseudo-random numbers - computers are just complex calculators, that can't create truly random numbers, but instead use an algorithm that generates numbers that seem random

Random numbers in JS

JavaScript provides a number of functions that are useful for generating random numbers

  1. Math.random()is a function that returns a pseudo-random number between 0 and 1, not including 1
  2. Math.floor() is a function that takes a decimal number and rounds it down - chops off the decimal part

Random Number Calculation

To produce a random number from min to max (including min and max) use this calculation.
Math.floor(Math.random() * (max - min + 1)) + min

Example
Generate a random number from 1 to 10 and store it in a variable.
var randNum = Math.floor(Math.random() * 10) + 1;
