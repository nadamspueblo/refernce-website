Notes

For Loops

For loops are used to repeat instructions a specific number of times.

Syntax 

for (var i = 0; i < max; i++){
  // Code to repeat
}
var i = 0; creates a counter variable i and starts it at zero i < max; is the condition. The loop will repeat until this condition is false. i++ increments the counter by 1
