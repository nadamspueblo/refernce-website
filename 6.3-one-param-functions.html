6.3 Functions with one parameter

Objectives

  • To be able to correctly define and call functions that have one parameter

Notes

Function review

Recall: functions are a way of grouping together a set of instructions with a name. We use functions to make code modular (reusable) and to increase the level of abstraction to make our coding projects easier to think about

Example function definition 

function myFunction() {
  // JS statements
}
Example object function (method) definition 
var myObject = {
  myFunction: function() {
    // JS statements
  }
}

Calling functions

To call (use or execute) the function, you just write then name of the function with () wherever you want execution to jump to your function to execute its code.

Example function call 
// Script statements before
myFunction();
// Script statements after
Example object function (method) call 
// Script statements before
myObject.myFunction();
// Script statements after

Parameters

Recall: parameters are how we get data into a function that it needs to do its work. Parameter values can be passed to the function as a literal value like "message" or in a variable

Example function calls with one parameter 

// "Important message" is a literal string value parameter for the alert() function
alert("Important message!");

// msg is a variable holding the value for the parameter in the console.log() function
var msg = "Log message";
console.log(msg);

Defining functions with parameters

To define a function with one parameter, you just need to put a variable name inside the () of the function definition

Syntax 

function myFunction(paramName){
  // JS statements
}
When you call this function, you must now supply a value for the parameter variable
myFunction("Value");
The parameter, paramName, will now contain the string "Value" to use inside the function.

Function Definition Example

In this example, we will write a function called doubleNumber() that will take one number as a parameter, double it, then print the result an alert or in the console. Then we will test it by calling the function with several different values for the parameter.

Problem #1 Square function

Write a function named square() that takes in a parameter and squares it. Squaring a number means multiplying it by itself. Then make several calls to that function in your start function to test it out. Does it work for all arguments?

Problem #2 Triple function

Write a function called triple() that triples a number and prints it out. Then in your start function, make several calls to the triple function to test it out.

Notice how the call to the function interrupts the sequential execution of statements. The program jumps down to the function and once the last statement in the function (or a return statement) has executed, the flow of control is returned to the point immediately following where the function was called.
